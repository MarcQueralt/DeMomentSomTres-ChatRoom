== Description ==

The DeMomentSomTres-ChatRoom uses OpenTok to provide online conferencing and web video conversations directly on your blog in order to engage readers.

**Features include:**

Video Chat using Video Embed functionality.

== Installation ==

You can download and install the DeMomentSomTres-ChatRoom using the built in WordPress plugin installer. 

Once installed, you will need to activate the DeMomentSomTres-ChatRoom plugin in the "Plugins" admin panel using the "Activate" link.
